# 0x15 - API

This simple project focuses on making a request to a RESTful API and saving its data to CSV and JSON (locally).

## Contents

- 0-gather_data_from_an_API.py: Given the employee ID, return the information of his/her TODO list progress.
  - Usage: `0-gather_data-from_an_API.py ID` (ID is an integer)
- 1-export_to_CSV.py: Given the employee ID, save the information of his/her TODO list progress and export it to CSV.
  - Usage: `1-export_to_CSV.py ID` (ID is an integer)
- 2-export_to_JSON.py: Given the employee ID, save the information of his/her TODO list progress and export it to JSON.
  - Usage: `2-export_to_JSON.py ID` (ID is an integer)
- 3-dictionary_of_list_of_dictionaries.py: Save the information of EACH employee TODO list progress and export it to JSON.
